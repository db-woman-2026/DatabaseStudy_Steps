# 데이터베이스 단계별 실습

[Windows 11 환경 준비](../windows-11.md)를 끝낸 뒤 개인 저장소의 `main`에서 1일차부터 순서대로 진행합니다. 코드와 쿼리는 직접 입력하고 실행 전 예상과 실행 후 검증을 함께 기록합니다.

## 전체 과정 자료

- [8일·48시간 과정 계획](./course-plan.md): 일차별 시간표, 결과물, 평가 기준
- [기초 읽기 자료](../basic/README.md): RDBMS, SQL, JSON, MongoDB 개념

## 실습 방식

1. `git branch --show-current`에 `main`이 표시되는지 확인합니다.
2. 오늘의 데이터 구조와 안전 범위를 먼저 읽습니다.
3. 기준 실행 결과를 확인합니다.
4. 결과를 예측하고 코드나 쿼리를 직접 입력합니다.
5. 처리 건수와 저장된 데이터를 다시 조회합니다.
6. 독립 실습과 평가를 마친 뒤 commit하고 push합니다.
