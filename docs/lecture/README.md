# DatabaseStudy_Steps Lecture

이 폴더에는 각 `step-N` 브랜치에서 실행할 상세 실습 문서가 누적됩니다.

`main`은 starter 브랜치이므로 먼저 `docs/basic/`을 읽습니다. 이후 `step-1`로 이동하면 첫 실습 문서가 추가되고, 각 다음 브랜치에는 이전 문서와 현재 단계 문서가 함께 들어 있습니다.

## 실습 방식

1. 현재 브랜치와 `git status`를 확인합니다.
2. 현재 단계 문서의 목표와 데이터 구조를 먼저 읽습니다.
3. `npm start`로 기준 결과를 확인합니다.
4. 연습 문제에 따라 코드를 조금씩 바꾸고 다시 실행합니다.
5. 생성된 데이터와 처리 건수를 조회해 결과를 검증합니다.

현재 브랜치에는 아직 단계별 실습 문서가 없습니다. `git switch step-1`부터 시작합니다.
