# 데이터베이스 단계별 실습

단계별 수업 자료입니다.

`main`에서는 `docs/basic/`을 먼저 읽습니다. `step-N` 브랜치에는 앞 단계 자료와 해당 단계 자료가 함께 들어 있습니다.

## 전체 과정 자료

- [8일·48시간 강의 계획](./course-plan.md): 전체 시간표, 일차별 산출물, 평가 기준, 학습 연결
- [강사용 운영 가이드](./instructor-guide.md): 60분 블록 운영, 질문 은행, 힌트와 평가 방법

각 `step-N.md`에는 설명, 시연, 실습, 예상 결과, 해설, 형성평가가 들어 있습니다.

## 실습 방식

1. 현재 브랜치와 `git status`를 확인합니다.
2. 현재 단계 문서의 목표와 데이터 구조를 먼저 읽습니다.
3. `npm start`로 기준 결과를 확인합니다.
4. 연습 문제에 따라 코드를 조금씩 바꾸고 다시 실행합니다.
5. 생성된 데이터와 처리 건수를 조회해 결과를 검증합니다.
